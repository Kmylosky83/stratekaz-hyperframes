# Compliance

Explicativos de normativa colombiana, organizados por norma.

| Carpeta | Norma | Marco legal |
|---|---|---|
| `sg-sst/` | Sistema de Gestión de SST | Decreto 1072/2015, Resolución 0312/2019, ISO 45001:2018 |
| `iso-9001/` | Sistema de Gestión de Calidad | ISO 9001:2015 |
| `iso-14001/` | Sistema de Gestión Ambiental | ISO 14001:2015, Decreto 1076/2015 |
| `iso-45001/` | Sistema de Gestión de SST | ISO 45001:2018 |
| `pesv/` | Plan Estratégico de Seguridad Vial | Resolución 40595/2022, Ley 1503/2011 |

Cada norma puede contener varios videos (intro, ciclo PHVA, requisitos, beneficios, casos prácticos).

Para contenido normativo preciso, apoyarse en las skills `anthropic-skills:*-colombia-experto`.
